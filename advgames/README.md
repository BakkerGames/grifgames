## Scott Adam's Collection

This folder contains the collection of Scott Adams games ported from the ScottFree version to the GRIF language.

Due to the design of GRIF, this is an almost perfect 1-to-1 conversion. Some extra GRIF values are added to allow them to run but nothing else is changed.

In the "maps" folder are ASCII maps for each game. Include the appropriate one with the game to see the room exit maps, such as `grif.exe adv01.grif maps\adv01_maps.grif`.

See the file "0readme.txt" for full information about these games including copyrights.
